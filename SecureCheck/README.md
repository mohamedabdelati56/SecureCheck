Basic Security Auditor
Developed by Mohamed Abdel-Aty.

Overview
SecureCheck is a Python-based tool designed for basic security auditing. It combines network reconnaissance (Port Scanning) and Data Loss Prevention (DLP) by scanning files for sensitive information.

Features
Network Scanner: Checks common ports (SSH, FTP, HTTP, etc.) to identify potential entry points.

Data Auditor: Uses Regex patterns to detect exposed PII (Emails, Credit Cards) in local files.
